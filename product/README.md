# README del producto — Candidate Intake Form

Aplicación frontend de intake de candidatos en 4 pasos.

## Prerequisitos
- Node.js 20+
- npm 10+

## Instalación

```bash
cd product/intake-form
npm install
```

## Variables de entorno
No requiere `.env` en esta fase.

## Cómo correr

```bash
cd product/intake-form
npm run dev
```

## Cómo buildear

```bash
cd product/intake-form
npm run build
```

## Cómo testear

```bash
cd product/intake-form
npm run lint
```

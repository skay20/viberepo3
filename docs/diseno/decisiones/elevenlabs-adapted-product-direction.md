# Direccion visual adaptada desde DESIGN.md para intake form

## Fecha
2026-04-10

## Contexto
`product/intake-form/DESIGN.md` fue importado con `npx getdesign@latest add elevenlabs`.
El objetivo no es copiar ElevenLabs, sino extraer principios útiles para una UI de recruiting ya existente.

La direccion visual anterior era premium/editorial y funcional, pero todavía cargaba una temperatura más cálida y una expresividad más de boutique que de producto técnico.

## Principios retenidos
- Contraste alto y limpio
- Superficies refinadas con sombras finas e inset sutil
- Jerarquía tipográfica fuerte
- CTAs sobrios y precisos
- Composición espaciosa, ordenada y silenciosa

## Principios descartados
- Waveforms, reproductores, demos o metáforas de audio/voz
- Layouts de marketing o secciones promocionales propias de un sitio brochure
- Copia de marca, claims, ilustraciones o patrones reconocibles de ElevenLabs

## Decision
La UI del intake adopta:
- shell exterior oscuro tipo chrome premium
- paneles internos claros y legibles
- acento cálido muy controlado
- tipografía sans ligera para títulos y cuerpo
- estados visuales sobrios, con prioridad de usabilidad sobre espectáculo

## Regla de prioridad
Si la guía visual y el producto real entran en conflicto:
1. usabilidad
2. accesibilidad
3. coherencia del dominio
4. referencia estilística

## Impacto
- La app se siente más contemporánea y técnica sin alterar el flujo funcional.
- `DESIGN.md` pasa a ser referencia visual explícita para futuras tareas de UI sobre este producto.

# Direccion visual premium editorial para intake form

## Fecha
2026-04-08

## Contexto
La primera version funcional de la UI resolvia flujo y validaciones, pero su tono visual era mas cercano a una app SaaS genérica que a una experiencia moderna, elegante y de alta gama.

El usuario pidio explícitamente buscar una direccion visual premium usando como referencia el repositorio `nextlevelbuilder/ui-ux-pro-max-skill`.

## Referencia utilizada
- Tipografia: `Classic Elegant` (`Playfair Display` + `Inter`)
- Color: `Luxury/Premium Brand`
- Lenguaje visual: mezcla controlada entre `Editorial Grid / Magazine` y `Liquid Glass`

## Decision
Se adopta una direccion visual:
- editorial y sobria
- premium con contraste alto
- acento oro controlado
- capas translúcidas muy moderadas
- motion suave de `240ms`

## Reglas
- No usar gradients chillones ni estetica AI morada por defecto.
- El acento oro debe funcionar como detalle de jerarquia, no como fondo dominante.
- El glass effect se usa solo como soporte de profundidad ligera; no debe afectar legibilidad.
- Los titulares pueden ser expresivos; los campos y formularios mantienen claridad operacional.

## Impacto
- Mejora la percepcion de calidad de la UI sin degradar accesibilidad base.
- Deja una base mas coherente para futuras pantallas de recruiting premium.
